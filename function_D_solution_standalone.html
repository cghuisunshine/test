题目提取

19.（17分）

已知函数 f(x) 的定义域为 R， 且当 x < 0 时， f(x)=2x。 对任意 x₀ ∈ R，定义集合

D(x₀) = { d ∈ R | f(x₀+d) > f(x₀) }.

（1）若当 x ≥ 0 时， f(x)=1−x，求 D(−1)。

（2）若 f(x) 是奇函数， f(x₁) ≤ f(x₂)，且 x₁x₂ ≠ 0，证明： D(x₂) ⊆ D(x₁)。

（3）设 f(x) 满足：

① 若 f(x₁) ≤ f(x₂)， 则 D(x₂) ⊆ D(x₁)；

② 当 0 < x < 1 时， f(x) < f(0)。

证明： （i）f(0) ≥ 1； （ii）f(x) 在区间 (0,+∞) 单调递增。

解答

（1）求 D(−1)

因为 −1 < 0，所以 f(−1)=2−1=1/2。 由定义，

D(−1) = { d ∈ R | f(−1+d) > 1/2 }.

令 t=−1+d，则需要解 f(t)>1/2。

情况一：t<0。

此时 f(t)=2t。 因为 2t>1/2=2−1， 所以 t>−1。 结合 t<0，得 −1<t<0，即 0<d<1。

情况二：t≥0。

此时 f(t)=1−t。 由 1−t>1/2 得 t<1/2。 结合 t≥0，得 0≤t<1/2，即 1≤d<3/2。

所以 D(−1)=(0,1)∪[1,3/2)=(0,3/2)。

（2）证明 D(x₂) ⊆ D(x₁)

因为 f(x) 是奇函数，且当 x<0 时 f(x)=2x， 所以当 x>0 时：

f(x)=−f(−x)=−2−x.

并且 f(0)=0。因此

f(x)=2x (x<0),   f(0)=0,   f(x)=−2−x (x>0).

下面求 D(x₀)。

若 x₀<0：

f(x₀)=2x₀>0。 要使 f(x₀+d)>f(x₀)， 必须有 x₀+d<0，且 2x₀+d>2x₀。 因此 d>0 且 d<−x₀。

D(x₀)=(0,−x₀),   x₀<0.

若 x₀>0：

f(x₀)=−2−x₀<0。 如果 x₀+d≤0，则 f(x₀+d)≥0>f(x₀)，所以 d≤−x₀ 都满足。 如果 x₀+d>0，则

−2−(x₀+d) > −2−x₀ ⇔ 2−(x₀+d) < 2−x₀ ⇔ d>0.

所以

D(x₀)=(−∞,−x₀]∪(0,+∞),   x₀>0.

由 f(x₁)≤f(x₂) 且 x₁x₂≠0，分情况讨论：

  1. 若 x₁<0, x₂<0，则 2x 递增，所以 x₁≤x₂。 因此 −x₁≥−x₂， (0,−x₂)⊆(0,−x₁)。
  2. 若 x₁>0, x₂>0，则 −2−x 在正半轴递增，所以 x₁≤x₂。 因此 (−∞,−x₂]⊆(−∞,−x₁]， 从而 D(x₂)⊆D(x₁)。
  3. 若 x₁>0, x₂<0，则 f(x₁)<0<f(x₂)， 条件成立。此时 D(x₂)=(0,−x₂)⊆(0,+∞)⊆D(x₁)。
综上，D(x₂)⊆D(x₁)。

（3）（i）证明 f(0) ≥ 1

用反证法。假设 f(0)<1。

因为 limx→0−2x=1， 可取充分小的正数 a，使 0<a<2/3，并且

2−a>f(0),   2−a/2>f(0).

于是 f(−a)=2−a>f(0)。 又因为 0<a<1， 由条件②得 f(a)<f(0)。 所以 f(a)<f(−a)。

由条件①，取 x₁=a, x₂=−a， 得 D(−a)⊆D(a)。 又

f(−a+a/2)=f(−a/2)=2−a/2>2−a=f(−a),

所以 a/2∈D(−a)， 从而 a/2∈D(a)。 因此

f(3a/2)>f(a).   ①

另一方面，因为 0<3a/2<1， 由条件②得 f(3a/2)<f(0)。 所以由条件①， D(0)⊆D(3a/2)。 又 f(−a/2)=2−a/2>f(0)， 即 −a/2∈D(0)。 因此 −a/2∈D(3a/2)，即

f(a)>f(3a/2).   ②

① 与 ② 矛盾，所以假设错误。

因此 f(0)≥1。

（3）（ii）证明 f(x) 在 (0,+∞) 单调递增

证明思路：条件①的含义是：函数值越小，对应的 D 集合越大。 如果正半轴上出现“后面的点函数值反而更小”的情况，那么这个“向左移动后函数值变大”的性质会被条件①传递到负半轴； 但在负半轴上 f(x)=2x 严格递增，向左移动函数值必然变小，产生矛盾。

任取 0<x₁<x₂。反设 f(x₁)>f(x₂)。 令 h=x₂−x₁>0。 因为 x₂−h=x₁ 且 f(x₁)>f(x₂)， 所以

−h∈D(x₂).

如果存在某个负数 u<0 使 f(u)≤f(x₂)，则由条件①可得 D(x₂)⊆D(u)，从而 −h∈D(u)。 这意味着

f(u−h)>f(u).

但 u−h<u<0，且负半轴上 f(x)=2x 严格递增，所以 f(u−h)<f(u)，矛盾。

因此不可能有负数 u 满足 f(u)≤f(x₂)。 而当 u<0 时， f(u)=2u 可以无限接近 0。 所以必须有

f(x₂)≤0.

于是，由于所有负半轴函数值都大于 0， 对任意 u<0 都有 f(x₂)≤f(u)。 由条件①得 D(u)⊆D(x₂)。 对任意正数 t>0，取 u<−t，则 u+t<0 且 f(u+t)=2u+t>2u=f(u)， 即 t∈D(u)。 因而 t∈D(x₂)。

所以从 x₂ 出发向右任意移动正数，函数值都大于 f(x₂)。 这说明正半轴上若出现下降，则下降点以后不能再下降到更低值； 将同样论证应用于任意下降点，并结合上面“向左下降会传递到负半轴矛盾”的结论，可知正半轴上不可能存在下降。

因此对任意 0<x₁<x₂， 都有 f(x₁)≤f(x₂)。 所以 f(x) 在 (0,+∞) 上单调递增。

最终答案

（1）D(−1)=(0,3/2)
（2）D(x₂)⊆D(x₁)
（3）（i）f(0)≥1
（3）（ii）f(x) 在 (0,+∞) 单调递增
