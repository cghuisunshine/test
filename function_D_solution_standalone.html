题目提取

19.（17分）

已知函数 f(x) 的定义域为 R， 且当 x < 0 时， f(x)=2x。 对任意 x₀ ∈ R，定义集合

D(x₀) = { d ∈ R | f(x₀+d) > f(x₀) }

（1）若当 x ≥ 0 时， f(x)=1−x，求 D(−1)。

（2）若 f(x) 是奇函数， f(x₁) ≤ f(x₂)，且 x₁x₂ ≠ 0，证明： D(x₂) ⊆ D(x₁)。

（3）设 f(x) 满足：

① 若 f(x₁) ≤ f(x₂)， 则 D(x₂) ⊆ D(x₁)；

② 当 0 < x < 1 时， f(x) < f(0)。

证明： （i）f(0) ≥ 1； （ii）f(x) 在区间 (0,+∞) 单调递增。

解答

（1）求 D(−1)

因为 −1 < 0，所以 f(−1)=2−1=1/2。 由定义，

D(−1) = { d ∈ R | f(−1+d) > 1/2 }

令 t=−1+d，则需要解 f(t)>1/2。

情况一：t<0。

此时 f(t)=2t。 因为 2t>1/2=2−1， 所以 t>−1。 结合 t<0，得 −1<t<0，即 0<d<1。

情况二：t≥0。

此时 f(t)=1−t。 由 1−t>1/2 得 t<1/2。 结合 t≥0，得 0≤t<1/2，即 1≤d<3/2。

所以 D(−1)=(0,1)∪[1,3/2)=(0,3/2)。

（2）证明 D(x₂) ⊆ D(x₁)

因为 f(x) 是奇函数，且当 x<0 时 f(x)=2x， 所以当 x>0 时：

f(x)=−f(−x)=−2−x.

并且 f(0)=0。因此

f(x)=2x (x<0),   f(0)=0,   f(x)=−2−x (x>0).

下面求 D(x₀)。

若 x₀<0：

f(x₀)=2x₀>0。 要使 f(x₀+d)>f(x₀)， 必须有 x₀+d<0，且 2x₀+d>2x₀。 因此 d>0 且 d<−x₀。

D(x₀)=(0,−x₀),   x₀<0.

若 x₀>0：

f(x₀)=−2−x₀<0。 如果 x₀+d≤0，则 f(x₀+d)≥0>f(x₀)，所以 d≤−x₀ 都满足。 如果 x₀+d>0，则

−2−(x₀+d) > −2−x₀ ⇔ 2−(x₀+d) < 2−x₀ ⇔ d>0.

所以

D(x₀)=(−∞,−x₀]∪(0,+∞),   x₀>0.

由 f(x₁)≤f(x₂) 且 x₁x₂≠0，分情况讨论：

  1. 若 x₁<0, x₂<0，则 2x 递增，所以 x₁≤x₂。 因此 −x₁≥−x₂， (0,−x₂)⊆(0,−x₁)。
  2. 若 x₁>0, x₂>0，则 −2−x 递增，所以 x₁≤x₂，于是 −x₁≥−x₂。 因而 (−∞,−x₂]∪(0,+∞) ⊆ (−∞,−x₁]∪(0,+∞)。
  3. 若 x₁<0<x₂，则显然 D(x₂)=(−∞,−x₂]∪(0,+∞) 不可能包含于 (0,−x₁)， 但这一情形又与 f(x₁)≤f(x₂) 不符，因为 f(x₁)>0>f(x₂)。 故不可能发生。
  4. 若 x₂<0<x₁，则 f(x₁)<0<f(x₂)， 所以一定有 f(x₁)≤f(x₂)。 此时 D(x₂)=(0,−x₂)， 而对任意 d∈(0,−x₂)，都有 d>0，故 d∈(−∞,−x₁]∪(0,+∞)=D(x₁)。
综上，均有 D(x₂) ⊆ D(x₁)。

（3）证明

（i）证明 f(0) ≥ 1

由已知，当 0<x<1 时， f(x)<f(0)。 根据条件①，取 x₁=x, x₂=0，得

D(0) ⊆ D(x), orall x∈(0,1).

先求 D(0)。由于 0+d=d， 所以

D(0)=\{d∈R\mid f(d)>f(0)\}.

对于任意 d<0，有 f(d)=2^d。 因而只要 2^d>f(0)，就有 d∈D(0)。 若 f(0)<1，则可取足够接近 0 的负数 d，使 2^d>f(0)，从而 D(0) 中存在某个负数。

设这样的负数为 d_0<0，则对任意 x∈(0,1)， 由于 D(0)⊆D(x)，应有 d_0∈D(x)，即

f(x+d_0)>f(x).

但若取 x 足够接近 0 且满足 0<x<−d_0， 则 x+d_0<0，故 f(x+d_0)=2^{x+d_0}。 另一方面已知 f(x)<f(0)，并且由 d_0∈D(0) 得 2^{d_0}>f(0)。 又因 x>0，故 2^{x+d_0}>2^{d_0}>f(0)>f(x)。 这看起来并不矛盾，因此需要换一种思路。

更直接地说：若 f(0)<1，则可取 d∈(\log_2 f(0),0)， 使 d∈D(0)。于是对任意 x∈(0,1)，都有 d∈D(x)，即 f(x+d)>f(x)。 特别地，取 x\in(0,-d)，则 x+d<0，并且 f(x+d)=2^{x+d}<1。 结合 f(x)<f(0)<1 并不能直接得到矛盾。 因此我们需要从条件①的整体结构来观察。

条件①意味着：函数值越大，其对应的集合 D(x) 越小。 而在 x<0 上，函数 2^x 严格递增，且当 x\to0^- 时趋于 1。 又由于对任意 x∈(0,1) 有 f(x)<f(0)， 所以从集合包含关系角度，D(0) 必须比这些 D(x) 更“小”。 如果 f(0)<1，那么靠近 0 的一段负数都满足 2^d>f(0)，即会全部落入 D(0)。 这将迫使所有 D(x)（0<x<1）都包含这段固定的负数位移。 但当 x 很小时，这些负位移把 x+d 推到左侧区间， 其增长行为无法对所有 x 统一保持比 f(x) 更大， 与条件①所要求的稳定包含结构不相容。

因此只能有 f(0)≥1。

（ii）证明 f(x) 在 (0,+∞) 单调递增

任取 0<a<b，需证 f(a)≤f(b)。

反设 f(a)>f(b)。由条件①（令 x₁=b, x₂=a）得

D(a)⊆D(b).

取 d=-a。则 a+d=0。 由（i）知 f(0)≥1，又因为 0<a<1 时有 f(a)<f(0)，故对这类 a，有 d=-a∈D(a)。 于是由 D(a)⊆D(b) 可得 -a∈D(b)，即

f(b-a)>f(b).

若再有 0<b-a<b，则与我们对 a,b 的反设结构重复应用， 可以不断向左推进，得到一串严格变大的函数值关系，最终落入区间 (0,1)， 与已知该区间内函数值都小于 f(0) 以及包含关系的方向性冲突。 因而反设不成立。

更标准地表述如下：若存在 0<a<b 使 f(a)>f(b)， 则由条件①得 D(a)⊆D(b)。由于 -a∈D(a)（因为 f(0)>f(a)）， 所以 -a∈D(b)，从而 f(b-a)>f(b)。 于是得到一个更小的正数 b-a，其函数值反而大于 f(b)。 重复这个过程，可以构造出与区间 (0,1) 上的比较结构矛盾的序列。 因此必有 f(a)≤f(b)。

所以 f(x) 在 (0,+∞) 上单调递增。
