一、题目1. Problem

题目原图：若 \(x^2+y^2=16\)，求根式表达式的值。Original image: Given \(x^2+y^2=16\), find the value of the radical expression.
\[ \text{若 }x^2+y^2=16,\quad \text{求 }\sqrt{y^2-8y+16}+\sqrt{xy-5y+4x-20}. \]

二、第一项化简2. Simplify the First Radical

\[ y^2-8y+16=(y-4)^2 \]
\[ \sqrt{y^2-8y+16}=|y-4| \]

又因为 \(x^2+y^2=16\)，所以 \(-4\le y\le4\)。因此：Since \(x^2+y^2=16\), we have \(-4\le y\le4\). Therefore:

\[ |y-4|=4-y \]

三、第二项的关键分解3. Key Factorization of the Second Radical

\[ xy-5y+4x-20 \]
\[ =xy+4x-5y-20 \]
\[ =(x-5)(y+4) \]

由 \(x^2+y^2=16\)，可知：From \(x^2+y^2=16\), we know:

\[ -4\le x\le4,\qquad -4\le y\le4 \]

所以：Thus:

\[ x-5\le -1\lt0,\qquad y+4\ge0 \]

因此：Therefore:

\[ (x-5)(y+4)\le0 \]

但它在根号里面，必须满足：But it is inside a square root, so it must satisfy:

\[ (x-5)(y+4)\ge0 \]

两者合起来，只能是：Together, this means it can only be:

\[ (x-5)(y+4)=0 \]

因为 \(x\le4\)，不可能 \(x=5\)，所以：Since \(x\le4\), it is impossible that \(x=5\), so:

\[ y+4=0 \]
\[ y=-4 \]

四、求出 \(x,y\) 并代入4. Find \(x,y\) and Substitute

把 \(y=-4\) 代入 \(x^2+y^2=16\)：Substitute \(y=-4\) into \(x^2+y^2=16\):

\[ x^2+(-4)^2=16 \]
\[ x^2=0 \]
\[ x=0 \]

原式为：The original expression becomes:

\[ \sqrt{(-4)^2-8(-4)+16}+\sqrt{0} \]
\[ =\sqrt{64}+0=8 \]
答案：8Answer: 8

五、最终答案5. Final Answer

\[ \boxed{8} \]
关键点：第二个根号不是随便取值，而是迫使 \((x-5)(y+4)=0\)，从而只能得到 \((x,y)=(0,-4)\)。Key point: the second radical forces \((x-5)(y+4)=0\), so the only possible point is \((x,y)=(0,-4)\).
