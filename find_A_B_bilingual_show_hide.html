一、题目1. Problem

题目原图：根据两个等式求 A 和 B。Original image: Use two equations to find A and B.

已知：Given:

A + B = 65 B / (A + 5) = 3 / 7

求：A = ____，B = ____Find: A = ____, B = ____

二、方法一：代入法2. Method 1: Substitution

由第一个等式：From the first equation:

A + B = 65

可以得到：We get:

B = 65 - A

把它代入第二个等式：Substitute this into the second equation:

(65 - A) / (A + 5) = 3 / 7

交叉相乘：Cross multiply:

7(65 - A) = 3(A + 5)

展开并解方程：Expand and solve:

455 - 7A = 3A + 15 440 = 10A A = 44

再求 B：Now find B:

B = 65 - 44 B = 21
A = 44，B = 21A = 44, B = 21

三、方法二：按比例理解3. Method 2: Understand the Ratio

由：From:

B / (A + 5) = 3 / 7

可知：We know:

B : (A + 5) = 3 : 7

设 1 份是 x，那么：Let one part be x, then:

B = 3x A + 5 = 7x

所以：So:

A = 7x - 5

又因为 A + B = 65：Since A + B = 65:

(7x - 5) + 3x = 65 10x - 5 = 65 10x = 70 x = 7

因此：Therefore:

B = 3x = 21 A = 7x - 5 = 44
A = 44，B = 21A = 44, B = 21

四、验算4. Check

A + B = 44 + 21 = 65 B / (A + 5) = 21 / (44 + 5) = 21 / 49 = 3 / 7

两个条件都满足，所以答案正确。Both conditions are satisfied, so the answer is correct.

最终答案：A = 44，B = 21Final answer: A = 44, B = 21
