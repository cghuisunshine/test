1. Start Here

C# is a programming language often used for web backends, desktop apps, games with Unity, cloud tools, and business software. It is strongly typed, which means variables have clear data types.

Main idea: In C#, you usually tell the computer what kind of data you are storing, such as int, double, string, or bool. 
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, C#");
    }
}

using System lets us use common built-in features like Console.

Main() is the starting point of a normal C# console program.

Console.WriteLine() prints text to the screen.

2. Variables and Data Types

Common types

int = whole number

double = decimal number

string = text

bool = true or false

Examples

int age = 16;
double price = 9.99;
string name = "Mia";
bool isReady = true;

You can also use var when C# can clearly figure out the type from the value.

var city = "Vancouver"; // string
var count = 3;          // int

3. Conditions: if / else

Conditions let your program choose what to do.

int age = 18;

if (age >= 18)
{
    Console.WriteLine("Adult");
}
else
{
    Console.WriteLine("Minor");
}

Comparison operators include ==, !=, >, <, >=, and <=.

4. Loops

Loops repeat code. Two common types are for and while.

for (int i = 0; i < 3; i++)
{
    Console.WriteLine(i);
}
int n = 1;
while (n <= 3)
{
    Console.WriteLine(n);
    n++;
}
A for loop is great when you know how many times to repeat. A while loop is useful when you repeat until a condition becomes false.

6. Arrays and foreach

An array stores multiple values of the same type.

string[] colors = { "red", "green", "blue" };

You can read an item by index. Arrays start at index 0.

Console.WriteLine(colors[0]); // red

You can also loop through them:

foreach (string color in colors)
{
    Console.WriteLine(color);
}

7. Classes and Objects

C# is an object-oriented language. A class is like a blueprint. An object is a real thing made from that blueprint.

class Dog
{
    public string Name;

    public void Bark()
    {
        Console.WriteLine("Woof!");
    }
}
Dog d = new Dog();
d.Name = "Coco";
d.Bark();
A class can contain data, such as Name, and behavior, such as Bark().

8. Final Challenge

Read the code and answer the questions.

using System;

class Program
{
    static int Square(int x)
    {
        return x * x;
    }

    static void Main()
    {
        int[] nums = { 2, 3, 4 };
        foreach (int n in nums)
        {
            if (n % 2 == 0)
            {
                Console.WriteLine(Square(n));
            }
        }
    }
}
