Addition and Subtraction with Parentheses

Practice only these two forms: a + (−b) and a − (−b). This page does not include a − (+b).

Rule 1: Adding a negative number is the same as subtracting that number. Example: 8 + (−3) = 8 − 3 = 5
Rule 2: Subtracting a negative number is the same as adding that number. Example: 8 − (−3) = 8 + 3 = 11
