第(3)问：图片解答是否有问题？（核对 + 解释）

目标：验证图片给出的结论是否正确，并说明为什么过程看起来“绕”。

结论先说：图片最终答案 m=8 或 m=26 是正确的。 我们用“代入验算 + 顶点跨越不可能”两步把它钉死。
小提示：你可以在页面里双击选中整段公式区（代码风格框）快速复制。

1) 已知桥拱函数（来自第1问）

桥拱是开口向下的抛物线：

f(x) = - (1/18) · (x - 18)^2

关键信息：

  • 顶点在 (18, 0)，最大值为 0。
  • 对称轴是 x = 18。
  • 当 x<18 时函数单调递增；当 x>18 时函数单调递减。

2) 第(3)问到底在求什么？

点：

E(m, y1), F(m+2, y2) 在桥拱上 ⇒ y1 = f(m), y2 = f(m+2)

曲线 G 是桥拱上从 x=m 到 x=m+2 的那一段（含端点）。

定义：

t = (G 上 y 的最大值) − (G 上 y 的最小值)

题目给：t = 2，求 m。

3) 核对图片答案：用代入验算“秒杀”对错

验算 m = 8

区间是 [8, 10]（在顶点 18 左侧，单调递增）。

f(8) = - (1/18) · (8-18)^2 = - (1/18) · 100 = -100/18 f(10) = - (1/18) · (10-18)^2 = - (1/18) · 64 = -64/18 左侧单调递增 ⇒ 最大值 = f(10), 最小值 = f(8) t = f(10) - f(8) = (-64/18) - (-100/18) = 36/18 = 2

t=2 成立OK

验算 m = 26

区间是 [26, 28]（在顶点 18 右侧，单调递减）。

f(26) = - (1/18) · (26-18)^2 = - (1/18) · 64 = -64/18 f(28) = - (1/18) · (28-18)^2 = - (1/18) · 100 = -100/18 右侧单调递减 ⇒ 最大值 = f(26), 最小值 = f(28) t = f(26) - f(28) = (-64/18) - (-100/18) = 36/18 = 2

t=2 成立OK

既然两组都严格满足 t=2，那么图片给出的最终答案 m=8 或 m=26 必然正确。

4) 为什么不会漏掉“跨过顶点”的情况？（解释最关键的判无解）

如果区间 [m, m+2] 包含顶点 x=18，则：

  • 最大值一定是顶点处的 0。
  • 最小值出现在离 18 更远的端点处（因为开口向下，离顶点越远越低）。

当包含顶点时，m 必须满足 16 ≤ m ≤ 18。这时区间长度只有 2，离 18 最远也就 2 个单位。

因此最小值最低只能到：

f(16) = - (1/18) · (16-18)^2 = - (1/18) · 4 = -4/18

于是：

t ≤ 0 - (-4/18) = 4/18 < 2

所以：只要区间跨过顶点，就不可能出现 t=2，因此这类情况必无解。 OK

这也解释了你在图片里看到的“算出某些 m 然后又舍去”的现象： 它们往往来自跨顶点或不满足分段范围的代数解，最终都必须被排除。

5) 最终结论

图片第(3)问的解答 没有问题；最终答案是：

m = 8 或 m = 26

如果你想把“过程写得更像标准满分答案”，最清爽的结构是： ① 判断区间是否包含顶点 → ② 若不包含，按单调性端点取极值 → ③ 列 t=2 解 m → ④ 检查分段条件。
