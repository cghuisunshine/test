一、题目1. Problem

题目原图：每个图形代表一个数字。 Original image: Each shape represents one digit.

已知：Given:

△○□ + △○○ = 777

求：△、○、□ 分别代表什么数字？ Find: the digit represented by △, ○, and □.

二、解法：按竖式从个位算起2. Solution: Use Column Addition from the Ones Place

设：Let:

△ = T ○ = C □ = S

原式可以写成：The equation can be written as:

TCS + TCC = 777

三、个位3. Ones Place

个位是：In the ones place:

S + C = 7 或 17

后面十位要产生进位，所以个位必须进 1：The tens place must create a carry, so the ones place must carry 1:

S + C = 17

四、十位4. Tens Place

个位进了 1 到十位：The ones place carries 1 to the tens place:

C + C + 1 = 17
2C + 1 = 17 2C = 16 C = 8
○ = 8

五、百位5. Hundreds Place

十位也进了 1 到百位：The tens place also carries 1 to the hundreds place:

T + T + 1 = 7
2T + 1 = 7 2T = 6 T = 3
△ = 3

六、求方形6. Find the Square

回到个位：Return to the ones place:

S + C = 17

因为 ○ = 8，所以：Since ○ = 8:

S + 8 = 17 S = 9
□ = 9

七、解法二：从百位、十位、个位倒推7. Method 2: Work from Hundreds, Tens, then Ones

补充解法图：先看百位，再看十位，最后看个位。 Additional solution image: Start from the hundreds place, then the tens place, then the ones place.

这个方法先从百位开始观察，因为结果的百位是 7，而两个加数的百位都是 △。 This method starts from the hundreds place, because the hundreds digit of the sum is 7, and both addends have △ in the hundreds place.

第一步：看百位Step 1: Hundreds Place

百位上是： In the hundreds place:

△ + △ + 进位 = 7

十位可能会向百位进 1，所以： The tens place may carry 1 to the hundreds place, so:

△ + △ + 1 = 7

因此： Therefore:

2△ + 1 = 7 2△ = 6 △ = 3
△ = 3

第二步：看十位Step 2: Tens Place

既然百位用了一个进位，说明十位相加一定超过 10。十位上是： Since the hundreds place received a carry, the tens-place sum must be at least 10. In the tens place:

○ + ○ + 个位进位 = 17

个位也会向十位进 1，所以： The ones place also carries 1 to the tens place, so:

○ + ○ + 1 = 17

因此： Therefore:

2○ + 1 = 17 2○ = 16 ○ = 8
○ = 8

第三步：看个位Step 3: Ones Place

个位上是： In the ones place:

□ + ○ = 17

因为 ○ = 8，所以： Since ○ = 8:

□ + 8 = 17 □ = 9
□ = 9

八、验算8. Check

把 △ = 3，○ = 8，□ = 9 代入原式： Substitute △ = 3, ○ = 8, and □ = 9 into the original equation:

△○□ = 389 △○○ = 388 389 + 388 = 777
最终答案：△ = 3，○ = 8，□ = 9 Final answer: △ = 3, ○ = 8, □ = 9
