方框代表几？What Number Does the Box Represent?

一、题目1. Problem

题目原图：上下两个方框代表同一个数。Original image: The two boxes represent the same number.

已知：Given:

(15 + □) / (80 + □) = 2 / 7

求：□ 代表多少？Find: the value of □.

二、方法一：设未知数2. Method 1: Use a Variable

设方框代表的数是 x。Let the box represent x.

(15 + x) / (80 + x) = 2 / 7

交叉相乘：Cross multiply:

7(15 + x) = 2(80 + x)

展开：Expand:

105 + 7x = 160 + 2x

移项：Move terms:

7x - 2x = 160 - 105 5x = 55 x = 11
□ = 11□ = 11

三、方法二：比例差值法3. Method 2: Ratio Difference Method

辅助理解图：把 2:7 看成 2 份和 7 份，它们相差 5 份；而分母与分子的实际差是 65，所以每份是 13。 Visual aid: View 2:7 as 2 parts and 7 parts. Their difference is 5 parts; the actual difference is 65, so each part is 13.

由题目可知：From the equation:

(15 + □) : (80 + □) = 2 : 7

分母比分子大：The denominator is larger than the numerator by:

(80 + □) - (15 + □) = 65

方框会抵消，所以差永远是 65。The boxes cancel, so the difference is always 65.

比例 2 : 7 的差是：The difference in the ratio 2 : 7 is:

7 - 2 = 5 份

所以 5 份对应 65：So 5 parts correspond to 65:

1 份 = 65 ÷ 5 = 13

分子是 2 份：The numerator is 2 parts:

15 + □ = 2 × 13 = 26

所以：Therefore:

□ = 26 - 15 = 11
□ = 11□ = 11

四、验算4. Check

(15 + 11) / (80 + 11) = 26 / 91 = 2 / 7

等式成立，所以答案正确。The equation is true, so the answer is correct.

最终答案：□ = 11Final answer: □ = 11
