How to Use This Tutorial

Trigonometry is easier when you keep track of three things: angle measure (degrees vs radians), where you are on the unit circle (signs by quadrant), and which model fits (right triangle, unit circle, or triangle laws).

Study routine (fast)

  1. Memorize special-angle values for sin and cos.
  2. Use tan = sin / cos (don’t memorize a separate table).
  3. Practice solving equations on an interval (e.g., 0 ≤ x < 2π).
  4. Do mixed problems: graphs, identities, and triangle solving.

Common mistakes

  • Mixing degrees and radians on a calculator.
  • Forgetting the sign in Quadrant II/III/IV.
  • Solving sin x = a with only one angle (usually there are two per cycle).
  • Using Law of Sines in an SAS case (use Law of Cosines first).

Calculator tip: set the mode explicitly. On most calculators: MODE → choose DEG or RAD.

1) Angles & Radians

Degrees vs radians

A full turn is 360° which equals 2π radians. Radians measure angles using arc length on a circle.

180° = π rad
radians = degrees × (π / 180)
degrees = radians × (180 / π)

Arc length and sector area

These formulas require radians.

s = rθ

s is arc length, r is radius, θ is in radians.

A = (1/2) r²θ

The sector is the “pizza slice” cut from the circle.

Angle direction and coterminal angles

  • Counterclockwise angles are positive; clockwise angles are negative.
  • Coterminal angles differ by multiples of 2π (or 360°).
θ and θ + 2kπ (k is any integer)

Quick conversion table

Degrees Radians Degrees Radians
30° π/6 180° π
45° π/4 270° 3π/2
60° π/3 360° 2π
90° π/2 720° 4π

2) Right-Triangle Trig (SOH–CAH–TOA)

In a right triangle, pick an acute angle θ. The sides are named relative to that angle: opposite, adjacent, hypotenuse.

Definitions (ratios)

sin θ = opposite / hypotenuse
cos θ = adjacent / hypotenuse
tan θ = opposite / adjacent

Reciprocal trig functions

csc θ = 1 / sin θ
sec θ = 1 / cos θ
cot θ = 1 / tan θ

Example 1 (solve for a side)

A right triangle has hypotenuse 13 and angle θ = 40°. Find the side opposite θ.

  1. Use sin θ = opposite / hypotenuse.
  2. sin 40° = opposite / 13.
  3. opposite = 13 sin 40° ≈ 8.36.

Round to the precision your course expects (often 3 s.f. or 1 decimal place).

Example 2 (find an angle)

In a right triangle, opposite = 7 and adjacent = 5. Find θ.

  1. tan θ = opposite / adjacent = 7/5.
  2. θ = arctan(7/5) ≈ 54.46°.
When using inverse trig in Grade 12, always state the mode (degrees or radians) and the interval if required.

3) Unit Circle (the “function” view)

The unit circle is a circle of radius 1 centered at the origin. For an angle θ in standard position, the point on the unit circle is:

(x, y) = (cos θ, sin θ)

This connects triangles to functions because as θ changes, sin θ and cos θ change smoothly.

Signs by quadrant (ASTC)

A quick sign rule (often remembered by “ASTC”):

Quadrant Angle range Positive trig functions
I 0 < θ < π/2 All
II π/2 < θ < π sin
III π < θ < 3π/2 tan
IV 3π/2 < θ < 2π cos

Reference angles

A reference angle is the acute angle between the terminal arm and the x-axis. It lets you use the special-angle values and then apply the quadrant sign.

QI: α = θ
QII: α = π − θ
QIII: α = θ − π
QIV: α = 2π − θ
Once you find α, compute sin α, cos α, tan α from special-angle triangles, then apply the correct sign for the quadrant of θ.

4) Special Angles (exact values)

These are the backbone of Grade 12 trig. You should be able to recall them without a calculator.

The two special triangles

45°–45°–90°

Side ratio: 1 : 1 : √2

sin 45° = √2/2, cos 45° = √2/2, tan 45° = 1

30°–60°–90°

Side ratio: 1 : √3 : 2 (opposite 30° is 1)

sin 30° = 1/2, cos 30° = √3/2, tan 30° = √3/3
sin 60° = √3/2, cos 60° = 1/2, tan 60° = √3

5) Graphs & Transformations

The basic trig graphs come from the unit circle. In Grade 12 you usually focus on the function form and transformations.

Base functions (key facts)

Function Period Range Notes
y = sin x 2π [-1, 1] Zeros at x = kπ.
y = cos x 2π [-1, 1] Max at x = 0.
y = tan x π All real numbers Vertical asymptotes at x = π/2 + kπ.

Transformation form (sin/cos)

Most courses use the form:

y = a sin(b(x − c)) + d
y = a cos(b(x − c)) + d
Parameter Meaning How to compute
a Amplitude (vertical stretch/flip) |a| is amplitude; if a < 0 then reflect across midline.
b Horizontal stretch (changes period) Period P = 2π / |b|.
c Phase shift (left/right) Shift is c to the right.
d Vertical shift (midline) Midline is y = d.

Example (analyze a sinusoid)

For y = 2 sin(3(x − π/6)) − 1:

  • Amplitude = 2.
  • Period P = 2π/3.
  • Phase shift = π/6 right.
  • Midline y = −1, range [-3, 1].

Transformation form (tan)

y = a tan(b(x − c)) + d

Period P = π/|b|. Asymptotes are shifted/scaled with b and c, then shifted vertically with d.

6) Core Identities (tools for simplifying & proving)

An identity is an equation that is true for all values where both sides are defined. The key skill is rewriting everything in terms of sin and cos.

Reciprocal and quotient identities

tan θ = sin θ / cos θ
cot θ = cos θ / sin θ
sec θ = 1 / cos θ
csc θ = 1 / sin θ

Pythagorean identities

sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ

The last two come from dividing sin²θ + cos²θ = 1 by cos²θ or sin²θ.

Even/odd and cofunction identities

sin(−θ) = −sin θ, cos(−θ) = cos θ, tan(−θ) = −tan θ
sin(π/2 − θ) = cos θ
cos(π/2 − θ) = sin θ
tan(π/2 − θ) = cot θ

Sum and difference (if included in your course)

sin(A ± B) = sin A cos B ± cos A sin B
cos(A ± B) = cos A cos B ∓ sin A sin B
Many Grade 12 courses focus mainly on reciprocal/quotient/Pythagorean identities and solving equations. If sum/difference identities are not in your syllabus, you can skip this box.

Example (prove an identity)

Prove: (1 − cos²θ) / sin²θ = 1 (where defined)

  1. Start with the left side: (1 − cos²θ) / sin²θ.
  2. Use sin²θ + cos²θ = 1 → 1 − cos²θ = sin²θ.
  3. Substitute: sin²θ / sin²θ = 1 (as long as sin θ ≠ 0).

7) Solving Trig Equations (on an interval)

Most Grade 12 questions ask for solutions in an interval, such as 0 ≤ x < 2π or 0° ≤ x < 360°.

Type A: basic (unit circle)

Solve sin x = 1/2 for 0 ≤ x < 2π.

  1. Find the reference angle: sin α = 1/2 → α = π/6.
  2. sin is positive in Quadrants I and II.
  3. Solutions: x = π/6 and x = 5π/6.

Type B: using identities

Solve 2 sin²x − 1 = 0 for 0 ≤ x < 2π.

  1. 2 sin²x − 1 = 0 → sin²x = 1/2.
  2. sin x = ± √(1/2) = ± √2/2.
  3. Reference angle is π/4.
  4. List solutions where sin x is positive and negative: x = π/4, 3π/4, 5π/4, 7π/4.

Type C: transformed trig (use a “core” angle)

Solve cos(2x) = 1/2 for 0 ≤ x < 2π.

  1. Let u = 2x. Then solve cos u = 1/2.
  2. In 0 ≤ u < 4π, solutions are u = π/3, 5π/3, 7π/3, 11π/3.
  3. Divide by 2: x = π/6, 5π/6, 7π/6, 11π/6.
When you substitute u = bx, remember the interval changes too (multiply the original interval by b before solving).

8) Inverse Trig Functions

Inverse trig functions undo trig functions, but only after restricting the domain so that the function is one-to-one.

Principal values (common convention)

Function Input range Output (principal) range
y = arcsin(x) [-1, 1] [-π/2, π/2]
y = arccos(x) [-1, 1] [0, π]
y = arctan(x) All real numbers (-π/2, π/2)

Example 1

Evaluate arcsin(−√2/2).

  1. We need an angle in [-π/2, π/2] whose sine is −√2/2.
  2. That angle is −π/4.

Example 2 (composition)

Evaluate sin(arccos(3/5)).

  1. Let θ = arccos(3/5). Then cos θ = 3/5 and θ ∈ [0, π].
  2. Think of a triangle: adjacent = 3, hypotenuse = 5 ⇒ opposite = 4 (by Pythagoras).
  3. sin θ = 4/5. Therefore sin(arccos(3/5)) = 4/5.

9) Solving Triangles (Law of Sines & Law of Cosines)

When the triangle is not right-angled (an oblique triangle), use the triangle laws. Use standard notation: sides a, b, c are opposite angles A, B, C.

Law of Sines

a / sin A = b / sin B = c / sin C

Best used for ASA, AAS, and sometimes SSA (ambiguous case).

Law of Cosines

a² = b² + c² − 2bc cos A
b² = a² + c² − 2ac cos B
c² = a² + b² − 2ab cos C

Best used for SAS and SSS.

Example (SAS)

In triangle ABC, let b = 7, c = 10, and A = 40°. Find a.

  1. Use Law of Cosines: a² = b² + c² − 2bc cos A.
  2. a² = 7² + 10² − 2(7)(10)cos 40°.
  3. a ≈ √(49 + 100 − 140·0.7660) ≈ 6.55.

Ambiguous case (SSA) idea

With Law of Sines, an SSA setup can create 0, 1, or 2 possible triangles because sin θ = sin(180° − θ).

If your course tests the SSA ambiguous case, always check whether the computed angle could also be 180° − angle, and whether the remaining angles stay positive.

10) Applications

Trigonometry models real-world situations: heights, distances, navigation, waves, and periodic motion.

Angle of elevation / depression

Angle of elevation is measured up from the horizontal. Angle of depression is measured down from the horizontal. They often form alternate interior angles with parallel lines, so they are equal in many diagrams.

If a question says “a person is 40 m from a building and looks up at 32°”, it’s usually a right triangle: tan 32° = height / 40.

Simple harmonic motion (sinusoidal model)

Many periodic phenomena can be modeled by y = a sin(b(x − c)) + d. The midline y = d is the average value, and the amplitude |a| is the maximum deviation from average.

Practice Set (Grade 12)

Try without a calculator first if an exact value is possible.

A) Radians & basics

  1. Convert 210° to radians.
  2. Convert 7π/6 to degrees.
  3. Find one positive coterminal angle for −5π/3.
  4. A circle has radius 12 cm. Find arc length for θ = 5π/6.

B) Special angles & unit circle

  1. Find exact sin(5π/6), cos(5π/6), tan(5π/6).
  2. Find exact csc(3π/4) and sec(7π/6).
  3. Find sin θ and tan θ if cos θ = −3/5 and θ is in Quadrant II.

C) Graphs & transformations

  1. For y = −3 cos(2(x + π/4)) + 1, state amplitude, period, phase shift, midline, and range.
  2. Find the period of y = tan(5x).

D) Identities & equations

  1. Simplify: (1 − sin²θ) / cos²θ.
  2. Solve sin x = −√3/2 on 0 ≤ x < 2π.
  3. Solve 2 cos²x − 3 cos x + 1 = 0 on 0 ≤ x < 2π.
  4. Solve tan(3x) = 1 on 0 ≤ x < 2π.

E) Inverse trig

  1. Evaluate arccos(−1/2).
  2. Evaluate cos(arctan(4/3)).

F) Solving triangles

  1. (SAS) In triangle ABC, b = 9, c = 11, A = 52°. Find a.
  2. (AAS) In triangle ABC, A = 35°, B = 75°, and a = 12. Find b and c.
  3. (SSA) A = 30°, a = 8, b = 10. Determine how many triangles are possible and find B (approx.).

Answer Key

A) Radians & basics
  1. 210° = 210 × (π/180) = 7π/6
  2. 7π/6 = 7 × 180/6 = 210°
  3. −5π/3 + 2π = −5π/3 + 6π/3 = π/3
  4. s = rθ = 12 × 5π/6 = 10π cm
B) Special angles & unit circle
  1. sin(5π/6)=1/2, cos(5π/6)=−√3/2, tan(5π/6)=−√3/3
  2. csc(3π/4)=1/(√2/2)=√2, sec(7π/6)=1/(−√3/2)=−2√3/3
  3. Quadrant II: sin θ > 0, tan θ < 0. With cos θ = −3/5, use sin²θ = 1 − cos²θ = 1 − 9/25 = 16/25, so sin θ = 4/5, and tan θ = (4/5)/(−3/5)= −4/3.
C) Graphs & transformations
  1. y = −3 cos(2(x + π/4)) + 1: amplitude 3, period P = 2π/2 = π, shift π/4 left, midline y = 1, range [1 − 3, 1 + 3] = [−2, 4].
  2. y = tan(5x) has period π/5.
D) Identities & equations
  1. (1 − sin²θ)/cos²θ = cos²θ/cos²θ = 1 (where cos θ ≠ 0)
  2. sin x = −√3/2 → reference angle π/3, negative in QIII and QIV: x = 4π/3, 5π/3.
  3. Let u = cos x: 2u² − 3u + 1 = 0 → (2u − 1)(u − 1)=0, so cos x = 1 or cos x = 1/2. On 0 ≤ x < 2π: x = 0, and x = π/3, 5π/3.
  4. tan(3x)=1 → 3x = π/4 + kπ. In 0 ≤ x < 2π, solutions are x = π/12, 5π/12, 9π/12, 13π/12, 17π/12, 21π/12 (i.e., add π/3 each time).
E) Inverse trig
  1. arccos(−1/2) = 2π/3 (principal range [0,π])
  2. If θ = arctan(4/3), then take a triangle with opposite 4, adjacent 3, hypotenuse 5. So cos θ = 3/5, hence cos(arctan(4/3)) = 3/5.
F) Solving triangles
  1. a² = 9² + 11² − 2(9)(11)cos 52° ⇒ a ≈ 8.69.
  2. C = 180° − 35° − 75° = 70°. By Law of Sines: b = a·sin B / sin A = 12·sin 75°/sin 35° ≈ 20.2, c = a·sin C / sin A = 12·sin 70°/sin 35° ≈ 19.6.
  3. Use Law of Sines: sin B = (b sin A)/a = 10·sin 30°/8 = 0.625. So B ≈ 38.68° or B ≈ 141.32°. Check: both can form a triangle because 30° + 141.32° < 180°.

Cheat Sheet (quick review)

Must-know formulas

sin²θ + cos²θ = 1
tan θ = sin θ / cos θ
s = rθ, A = (1/2)r²θ (θ in radians)
Law of Sines: a/sin A = b/sin B = c/sin C
Law of Cosines: a² = b² + c² − 2bc cos A

Solving checklist

  1. What model? Right triangle, unit circle, or triangle laws?
  2. What mode? Degrees or radians?
  3. What interval? (e.g., 0 ≤ x < 2π)
  4. What quadrant/sign?
  5. Do you need exact or approximate answers?
