Bond Order in Chemistry

Bond order is a compact way to describe how strongly two atoms are connected: higher bond order usually means a shorter, stronger bond. This page shows how to calculate bond order using both Lewis structures (including resonance) and molecular orbital (MO) theory, with worked examples and practice.

1) Definition

Bond order is the number of bonding “connections” between two atoms. In simple Lewis drawings, it often matches what you draw:

  • single bond → bond order 1
  • double bond → bond order 2
  • triple bond → bond order 3
Important: bond order can be fractional (like 1.5) when electrons are delocalized (resonance) or when MO theory predicts partial bonding.

2) Why it matters

Bond strength (bond energy)

Higher bond order usually means a larger bond energy (harder to break). For the same pair of atoms, triple bonds are typically stronger than double bonds, which are stronger than single bonds.

Bond length

Higher bond order usually means a shorter bond length because more electron density is concentrated between the nuclei.

Caveat: bond order is not the only factor. Atom size, electronegativity differences (ionic character), and the surrounding molecular environment can change bond lengths and energies.

3) Bond order from Lewis structures

For many introductory problems (especially for discrete molecules/ions with clear single/double/triple bonds), you can estimate bond order directly from a Lewis structure.

Procedure (Lewis approach)

  1. Draw a reasonable Lewis structure (or set of resonance structures).
  2. Count the number of bonds between the two atoms of interest.
  3. If resonance applies, average over equivalent positions (next section).

Example

O=O   → bond order = 2
N≡N   → bond order = 3
H–H   → bond order = 1
What Lewis bond order is really saying: it’s a bookkeeping shortcut for shared electron pairs. It works best when electrons are localized and the structure is dominated by a single bonding pattern.

4) Fractional bond order (resonance)

When a molecule has multiple valid resonance structures, the real electron distribution is a resonance hybrid. Bonds that alternate between single and double in different resonance forms often end up with a bond order between 1 and 2.

How to compute a resonance-averaged bond order

For a set of equivalent bonds, add up the bond orders in each resonance form and divide by the number of resonance forms (or, more generally, use weighted averaging if forms are not equally important).

System Equivalent bonds Resulting bond order Why
benzene (C–C) 6 1.5 alternating single/double in two main resonance forms
nitrate, NO3− (N–O) 3 4/3 ≈ 1.33 one N=O and two N–O in each form, rotated among O atoms
carbonate, CO32− (C–O) 3 4/3 ≈ 1.33 same “one double + two single” pattern over 3 positions
Note: resonance-averaged bond order is a model. Experimental bond orders can be extracted from bond lengths and spectroscopic data, and may deviate from simple fractions.

5) Bond order from molecular orbital (MO) theory

MO theory treats electrons as occupying molecular orbitals spread over the molecule. For diatomic molecules (and many simple cases), it gives a powerful quantitative definition of bond order:

MO bond order formula: bond order = (Nbonding − Nantibonding) / 2

What counts as bonding vs antibonding?

  • Bonding MOs place electron density between nuclei (stabilize).
  • Antibonding MOs reduce density between nuclei (destabilize), often labeled with an asterisk, e.g., σ*, π*.

Procedure (MO approach, typical diatomic problems)

  1. Count total electrons (often valence electrons for simplified MO diagrams).
  2. Fill MOs from low to high energy, using Pauli exclusion and Hund’s rule.
  3. Count electrons in bonding MOs and antibonding MOs.
  4. Apply the formula above.
MO ordering warning (2nd-period homonuclear diatomics): the relative order of σ(2p) and π(2p) differs for B2, C2, N2 vs O2, F2. Always use the ordering your course expects.

6) Worked examples

Example A: H2 vs He2

H2 has 2 electrons.

σ(1s): 2 e− (bonding)
σ*(1s): 0 e− (antibonding)
Bond order = (2 − 0)/2 = 1

Conclusion: stable single bond.

He2 has 4 electrons.

σ(1s): 2 e− (bonding)
σ*(1s): 2 e− (antibonding)
Bond order = (2 − 2)/2 = 0

Conclusion: no net bond (not stable as a molecule under normal conditions).

Example B: O2 and its ions (bond order and magnetism)

O2 is a classic case where MO theory explains experimental observations: it is paramagnetic (attracted to a magnet) because it has unpaired electrons in π* orbitals.

Species Change in electrons Where electron is added/removed Bond order trend
O2 baseline π* partly filled bond order 2
O2+ −1 e− remove from π* bond order increases (stronger/shorter)
O2− +1 e− add to π* bond order decreases (weaker/longer)
O22− +2 e− fill π* bond order 1 (peroxide)

Example C: NO (heteronuclear diatomic)

NO has an odd number of electrons, so it is a radical. A common simplified MO treatment gives a bond order of about 2.5 and predicts paramagnetism (one unpaired electron).

Takeaway: fractional bond orders naturally appear in MO theory, not just resonance.

Example D: Benzene (resonance)

In benzene, each C–C bond is equivalent. Lewis resonance forms alternate single and double bonds, so each bond averages to 1.5. Experimentally, all C–C bond lengths are the same and intermediate between typical single and double bonds.

7) Trends & interpretations

Relationship summary

  • bond order ↑ → bond length ↓ (usually)
  • bond order ↑ → bond energy ↑ (usually)
  • antibonding electrons ↑ → bond order ↓

Quick reference

Bond order Typical description Relative length Relative strength
0 no net bond — —
1 single longer weaker
2 double shorter stronger
3 triple shortest strongest

8) Common pitfalls

  • Mixing models blindly: Lewis bond orders are bookkeeping; MO bond orders come from orbital occupancy. They often agree, but not always (e.g., O2 magnetism).
  • Resonance weighting: not all resonance structures contribute equally if some have higher formal charges or break octets; an unweighted average can be an oversimplification.
  • Coordinate bonds: a dative bond is still a bond; in many contexts it counts as bond order 1, but real bonding can have partial ionic character.
  • “Hypervalent” species: expanded-octet drawings can be misleading; MO or modern bonding descriptions may distribute electron density differently than a simple “extra bonds” picture.
  • Bond order vs formal charge: formal charge helps pick good Lewis structures; it is not the same as bond order.

9) Practice problems

Try these without looking, then open the solution.

Problem 1: What is the bond order of each N–O bond in NO3−?

There are 3 equivalent N–O bonds. Each resonance form has one N=O (bond order 2) and two N–O (bond order 1), so the total across the three bonds is 2 + 1 + 1 = 4. Average: 4/3 ≈ 1.33 for each N–O bond.

Problem 2: Use MO theory to predict whether He2+ has a bond. What is its bond order?

He2+ has 3 electrons total. Fill: σ(1s) gets 2 (bonding) and σ*(1s) gets 1 (antibonding). Bond order = (2 − 1)/2 = 0.5, so it has a weak net bond.

Problem 3: Compare the bond order of O2, O2−, and O2+.

In the common MO picture, extra electrons go into antibonding π* orbitals. Adding an electron (O2−) decreases bond order by 0.5 relative to O2; removing an electron (O2+) increases bond order by 0.5. So O2+ > O2 > O2−.

Problem 4: What is the average C–C bond order in benzene?

Benzene’s C–C bonds are equivalent due to resonance. In a simple resonance average, each C–C bond has bond order 1.5.

Tip: If you’re doing a course problem, use the model your course specifies (Lewis vs MO), show your electron counting clearly, and state assumptions (especially MO ordering rules).
